# Code of Conduct

## Purpose

This Code of Conduct outlines expectations for behavior and interaction within our project. It aims to ensure a positive and productive environment for all contributors.

## Expectations

1. **Professionalism**: Communicate and collaborate in a professional manner. Be respectful and considerate in all interactions.

2. **Constructive Feedback**: Provide and receive feedback in a constructive, supportive way. Focus on ideas, not individuals.

3. **Responsiveness**: Respond to messages, issues, and pull requests in a timely manner to keep the project moving forward.

4. **Ownership**: Take responsibility for your contributions and address any issues or concerns promptly.

5. **Privacy**: Respect the privacy of others. Do not share personal information without permission.

## Enforcement

Any behavior that disrupts the project or negatively impacts the experience of others will be addressed. If you encounter issues or have concerns, please contact the project maintainers.
